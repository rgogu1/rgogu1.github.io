# Portfolio (Static Site)

This is a self-contained personal portfolio website (no build step).

## Edit Content

Update `content.json`:
- Name, role, location
- About text
- Projects
- Experience
- Links (GitHub, LinkedIn, email)

## Run Locally

From this folder:

```sh
cd /Users/mr.ram/portfolio
python3 -m http.server 5173
```

Open `http://localhost:5173`.

## Deploy

Any static host works (GitHub Pages, Netlify, Vercel static, Cloudflare Pages).
Upload the folder contents as-is.
